Vim基本操作

进入插入模式：a或i

进入控制模式：ctrl+c或esc

光标移动：hjkl（左上下右）

多行移动：n+hjkl

删除一行：dd

删除多行：n+dd

撤销操作：u

撤销撤销操作：ctrl+r

到本文第一行：1G

到本文最后一行：G

向下翻页：ctrl+f

向上翻页：ctrl+b

先写到这里，常用的应该够了，以后不够再继续学。

语法高亮。

编辑_vimrc文件加入以下内容：

set nu!

colorscheme desert

syntax enable

syntax on

这些设置使得gvim可以显示行号，并使用了desert配色方案，而且打开了语法高亮功能（用不同颜色显示注释、关键字、字符串等）。

PS：其他的，貌似还有用插件在程序中函数定义跳转，不过现在看的少，等以后觉得需要了再加。

 

=====咯咯咯=====

统计字数和行数，按Ctrl+m即可。

"字符统计函数

if !exists("*CalCharCount")

function CalCharCount()

exe '%s/\S/&/gn'

endfunction

endif

"映射命令模式下ctrl+m调用字符统计

nnoremap <C-m> :call CalCharCount()<cr>
